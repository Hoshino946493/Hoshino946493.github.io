Termux PRoot 容器封装脚本

轻量化 Bash 脚本，封装复杂 PRoot 启动参数，多命令模式管理容器，兼容全版本 Android，开箱即用。

快速安装 使用文档 更新日志

核心特性

多命令模式

支持指定目录启动、容器名快捷登录、精简别名，同时兼容传统路径调用方式。

全安卓版本适配

自动检测 Android 版本，为 Android 8.0 以下系统自动生成 DNS 配置，解决网络问题。

彩色输出

所有提示信息都有颜色区分，信息类型一目了然，使用体验更友好。

中英文双语

首次运行可选择语言，支持中文和英文，语言设置持久化保存。

预置 PRoot 参数

内置挂载 /dev /sys /proc /sdcard /storage 等目录，环境变量自动配置。

统一容器目录

默认容器存放于 ~/.vroot/containers，集中管理所有 RootFS 镜像。

安装部署 

# 克隆仓库
git clone https://github.com/Hoshino946493/VRoot.git
cd VRoot

# 添加执行权限
chmod +x vroot

# 全局安装（推荐，任意目录可直接调用 vroot）
cp vroot $PREFIX/bin/

安装完成后，输入 vroot help 查看帮助文档。

使用说明

命令总览

msrfs <路径> [用户名]手动指定 RootFS 目录并登录容器，默认用户 root
login / l <容器名> [用户名]快捷登录 ~/.vroot/containers 下的容器，l 为简写
help / h查看帮助信息
<目录路径> [用户名]兼容旧用法：直接传入目录启动容器

使用示例 

# 手动指定路径登录
vroot msrfs ~/ubuntu-fs sensei

# 快捷登录容器
vroot login ubuntu-fs
vroot l debian-fs root

# 兼容旧用法
vroot ~/alpine-fs

# 查看帮助
vroot help

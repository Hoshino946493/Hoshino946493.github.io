# Hoshino946493's GitHub Pages

A simple command-line calculator written in C.

## Calculator - Quick Start

### Compile

```bash
gcc calc.c -o calc
```

Run

```bash
./calc
```

Example

```bash
./calc
输入算式(Enter mathematical expression):3.5 + 2.1
结果(Result):5.60
```

Error Handling

Situation Error Message
Division by zero 错误：除数不能为0(Error: Divisor cannot be zero.)
Unsupported operator 不支持的运算符(Unsupported operator.)

Requirements

· C99 or later
· Linux / macOS (uses system("clear"))
  · For Windows: change clear to cls
